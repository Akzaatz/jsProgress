Une propriété est une variable qui appartient à un objet. (permet de récupérer des valeurs)
Une méthode est une fonction qui appartient à un objet en particulier (fonctionnalité qui permet de faire une action)

  • unshift : ajoute un élément en début de tableau
  • push : ajoute un élément en fin de tableau
  • shift : supprime un élément en début de tableau
  • pop : supprime un élément en fin de tableau

Opérateur Typeof

L’opérateur Typeof (opérateur unaire : n’a besoin que d’une seule expression pour fonctionner) Permet de récupérer le type de la valeur qu’on lui passe en paramètre.

console.log(typeof VALEUR OU IDENTIFIANT)

Objets littéraux

Un littéral d'objet est une constantes possédant des propriétés.

Une propriété est un identifiant qui va être lié à une valeur : c’est une variable qui appartient à un objet.

Une méthode est une fonction qui appartient à un objet.

const Objet = {
propriété : 'valeur', (virgule pour séparer les propiétés jusquà la fin de la liste de propiétés)
propriété : 'valeur',
propriété : 'valeur'
}

Pour changer une valeur :

objet.propiété = nouvelle valeur

pour acceder à un objet :

objet. propriété()
OU
objet.méthode(valeur)

Notation Class & constructeur

contrairement au littéral d'objet qui est unique, les class permettent de créer des collections d'objets :
